# Модуль для внедрения rel=canonical
## Общий функционал
Cлужит для указания каноничной страницы через <link rel="canonical" href="ссылка"> в head
## Установка модуля
Модуль устанавливается стандартным способом. После установки создается HL-блок AffettaCanonical
## Полезная информация
### Описание полей HL-блока:
- Активность - в выдаче участвуют только активные страницы
- ***Внешний код - это поле предусмотрено для будущего функционала модуля (выгрузка данных из файла), пока он в работе.***
- URL - ссылка текущей страницы, для которой будет прописан rel="canonical"
- Каноническая ссылка - ссылка, которая будет добавлена в href (< link rel="canonical" href="ссылка" >)
